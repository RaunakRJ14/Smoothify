## üöÄ Getting Started
Welcome to Smoothify, a powerful tool that helps you transform jagged polygons and lines into smooth and visually appealing features. It uses an optimized version of Chaikin's corner-cutting algorithm to refine geometries derived from raster data classifications.

This guide will help you download and run Smoothify quickly and easily.

## üéÆ Using Smoothify
Once you have the application open, follow these simple steps to refine your geometries:

1. **Upload Your Data:**
   - Click on "Upload". Select the raster data file you want to smooth.

2. **Adjust Parameters:**
   - Use the settings to adjust the smoothing, if needed. You can typically set levels of smoothing to customize your output.

3. **Process the File:**
   - Hit the "Smooth" button. Wait for the process to complete. This usually takes a few moments, depending on the file size.

4. **Save Your Results:**
   - After processing, click "Save". Choose a location on your computer to store the smoothed output.
